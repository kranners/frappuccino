# Boolean operations Ruby

Use these operators:
```shell
AND = apple && banana

OR = apple || banana
```

NOT these operators:
```shell
apple and banana

apple or banana
```

See [The Ruby Language operator precedence](https://phrogz.net/ProgrammingRuby/language.html#table_18.4)
