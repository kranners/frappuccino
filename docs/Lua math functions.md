# Lua math functions

Lua's `math` is a table which serves as an interface to the standard C math library.

```lua
math.cos(x)
math.floor(y)
```

See [Mathematical Functions from the Lua documentation](https://www.lua.org/manual/5.1/manual.html#5.6)
