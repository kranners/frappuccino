# The `Array` type

The shorthand for `T[]` is _identical_ to `Array<T>`:

```typescript
type Basket = Fruit[];

// Is the same as
type Basket = Array<Fruit>;
```

[See the array type in the TypeScript documentation](https://www.typescriptlang.org/docs/handbook/2/objects.html#the-array-type)
